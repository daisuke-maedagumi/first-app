## Git Githubのルールについて

作業の際は必ずブランチを切って作業を行う。

Masterで作業は行わない。行う際は他の作業者に確認を行うこと。

ブランチ名は簡潔に作業内容がわかりやすい命名にする。

Pushする際は必ずプルリクエストを立てること。

## 質問について
